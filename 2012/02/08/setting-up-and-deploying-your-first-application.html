Reading time : Less than 10 minutes This is article is for newbies who want to create and deploy their first OpenShift application with RedHat cloud service. OpenShift is the code name of the Red Hat’s platform as a service PaaS for PHP, python (and more) applications to run on RedHat infrastructure. OpenShift client tools are ruby modules so the one who start working with OpenShift need to have Ruby support on their machines. I used Ubuntu 11.10 for setting the development system up. Lets start by installing the per-requisites.

sudo apt-get updatesudo apt-get install ruby ruby-dev rubygems git

Once the installation is completed, we can start installing client tools. All these tools are gems. Use the following command,

sudo gem install rhc

This command will install all client tools required to create and manage your OpenShift applications. Now, lets sign-up with OpenShift. You need a login id and password for creating and managing applications on OpenShift. Once you have created an account, you can move back and get a terminal. The URL for any application hosted on OpenShift will be looking like,

http://-.rhcloud.com/

First of all, create a domain for hosting your applications. Once, a domain is ready, you can create an application also. Remember, you can have one domain and any number of applications with RedHat cloud.

rhc-create-domain -n -l rhc-create-app -a -t php-5.3 -l

I have created a PHP application with the above command. You can run other types of applications as well. The second command will generate a simple code base and a git repository in the current directory. Move to the directory and browse through the code base. The exposed PHP files are located in php/ and you can change or add files in it. Open index.php and make it look exactly like the code below. https://gist.github.com/5076132 The code simple, it just makes a call to the phpinfo() method to make sure PHP works fine. Now, commit these changes using git and push it to the master branch on the RedHat cloud.

git commit -am “My first change in the application”git push

The application is ready to run now. You can access your application URL now using any browser. Remember, it will be looking like -.rhcloud.com. Take a look at my application here : http://redshift-subinsebastien.rhcloud.com/ //See you in the afterlife//Subin Sebastian
