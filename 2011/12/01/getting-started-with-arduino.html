Reading Time : Less than 10 minutes In this article I’m trying to introduce you a simple microcontroller board. There is nothing surprising for those people who are using Arduino already, since I just use an example program which is bundled with the Arduino software. So, only those noobs can come along, and others, please google some other interesting articles. By the way, you can go and get started with Arduino stuff from here. There are a number of Arduino boards are available in the market and there are a couple of clones are also available. I’m using the one called Duemilanove which is a property of Emdesic. The board’s name is pretty easy to pronounce, huh? Take a look at it.

If you are using Debian based Linux distro, there is a package named arduino available in the repository. You can install the application using the following command. Windows users can use the installer available here.

sudo apt-get install arduino

With this application there are a number of examples are bundled. Lets open one of the simple ones out of it. File -> Examples -> Basics -> Blink.

The two functions named setup() and loop() should be there by default. The Arduino make will be looking for those methods in the application. In the loop() function, you will see a simple logic. I’m extending it to something like this.

void loop() {  digitalWrite(13, HIGH);  digitalWrite(12, LOW);  digitalWrite(11, LOW);  digitalWrite(10, LOW);  delay(100);  digitalWrite(13, LOW);  digitalWrite(12, HIGH);   digitalWrite(11, LOW);   digitalWrite(10, LOW);  delay(100);  digitalWrite(13, LOW);  digitalWrite(12, LOW);   digitalWrite(11, HIGH);   digitalWrite(10, LOW);  delay(100);  digitalWrite(13, LOW);  digitalWrite(12, LOW);   digitalWrite(11, LOW);   digitalWrite(10, HIGH);  delay(100);}

This is pretty cumbersome I know. I’m lazy enough to reduce the number of lines of code! You can replace it with some bit shift operations or a switch or whatever. Now, time to get some LEDs. I use normal 4mm dia red color LEDs which is really standard to use. One thing is that, these kind of LEDs are brighter than the transparent glass type ones, even in lower voltages.

Get a bread board and wire the ground (GND pin 13) to a common connector. Then wire all these LEDs between each pins till pin 10 and the ground. Now, press the Upload button to move your compiled binaries to the board and it will start running automatically. You can check the activity of the program upload with the onboard spot LEDs. //See you in the afterlife//Subin Sebastian
