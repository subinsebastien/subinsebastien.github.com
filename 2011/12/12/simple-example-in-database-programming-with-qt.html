Most of the real world applications use databases to store data in a structured manner. Databases provide a very programmatic way to store and retrieve organised data. Qt, just like any other professional toolkit, provides rich set of database drivers and libraries to handle database functionality in a very managed way. It this, post, I would like to show you a simple example of how to use database with Qt. To begin with, create a console qt application and name it, say DBApp. Then add a reference to the project(.pro) file that we are going to use database in this project.

QT       += sql

You might have to rescan the project folder to refresh the changes. Now we have to define a database object and can use queries to fetch data from the database. For that, we have to create a database first. In this example, I’ve used MySQL database server for storing data. If you wish you can use other database servers as well. https://gist.github.com/5075241 Now, the database is ready to be used with our application. Now take a look at the code. https://gist.github.com/5075227 Though the code is pretty self descriptive, I would like to explain how it works. First we create a QSqlDatabase object using the addDatabase() function. The parameter to this function, specifies what type of Qt driver has to be used for this connection. The following few lines, sets the database options like username, password etc. Once it is opened using the function open() declare a QSqlQuery object named qry. The query used to fetch all data from the table is,

select * from fruit;

So, simply fetch the data from the database and print it to the debug console using the function, qDebug(). So, that’s all folks, you can look for more advanced examples in the Qt documentation here. Take a look at the output.
